## 🚀 Overview

A production-ready, responsive agency portfolio site for **Linfy Tech** — targeting tech startups with a professional and modern tone.

| Attribute | Value |
|---|---|
| **Type** | Agency portfolio site |
| **Primary colour** | `#0F172A` (dark slate) |
| **Accent colour** | `#6366F1` (indigo) |
| **Copy tone** | Professional & modern |
| **Target audience** | Tech startups |

## 🎨 Design System

- **Dark bg**: `#0F172A` · **Accent**: `#6366F1` (indigo) · **Hover**: `#4F46E5`
- **Font**: Inter (Google Fonts) — weights 300–900
- **Framework**: Tailwind CSS v3 via CDN — zero build step
- **Icons**: Heroicons (inline SVG)
- **JS**: Vanilla ES2020, no dependencies
